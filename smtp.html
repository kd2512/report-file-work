SMTP Protocol

SMTP (Simple Mail Transfer Protocol) is a protocol used for sending and receiving emails. It operates over the TCP/IP protocol suite and is responsible for the transmission of electronic mail between mail servers.

Key Features of SMTP

  • Text-based protocol for communication between mail servers.
  • Uses port 25 for communication.
  • Relies on other protocols like POP3 and IMAP for email retrieval by clients.

SMTP Commands

SMTP uses various commands for communication between the email client and server. Some common commands include:

Command Description
EHLO Identifies the sending server to the receiving server.
MAIL FROM Specifies the sender's email address.
RCPT TO Specifies the recipient's email address.
DATA Indicates the start of the actual message data.
